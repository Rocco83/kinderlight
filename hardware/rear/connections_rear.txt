# KinderLight RGB - Rear PCB - Connections Map

# --- Power Connections ---
External 12V Input        → Buck Converter VIN+
External GND              → Buck Converter GND

Buck Converter 5V Output  → ESP32 VCC
Buck Converter GND Output → ESP32 GND

Buck Converter 5V Output  → J3 Pin1 (5V) (toward Front PCB)

# --- Serial Interface (Front to Rear) ---
J3 Pin1 (GND)             → GND
J3 Pin2 (TX Front)        → ESP32 RX
J3 Pin3 (RX Front)        → ESP32 TX
J3 Pin4 (DTR Front)       → ESP32 EN (Reset)
J3 Pin5 (RTS Front)       → ESP32 BOOT (GPIO9)

# --- ESP32 Internal Connector ---
J4 Pin1 (GND)             → ESP32 GND
J4 Pin2 (TX)              → ESP32 TX
J4 Pin3 (RX)              → ESP32 RX
J4 Pin4 (DTR)             → ESP32 EN
J4 Pin5 (RTS)             → ESP32 GPIO9 (BOOT)

# --- Battery Backup (Optional) ---
Battery + (Optional)      → Diode (optional) → VCC
Battery - (Optional)      → GND

# --- Tag-Connect (Optional) ---
Pads for Tag-Connect TC2030 footprint mapped to:
- GND
- TX
- RX
- EN (Reset)
- GPIO9 (Boot)
